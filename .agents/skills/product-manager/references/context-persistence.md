# Context Persistence

Use `current-state.md` as the live source of truth for active work.

Use `index.md` as the navigation layer so the latest active artifacts are easy to find without searching.

Update `current-state.md` when facts, decisions, scope, blockers, or next actions change.

Update `index.md` when a major artifact is created, when active work changes, or when a different artifact becomes the best resume point.
